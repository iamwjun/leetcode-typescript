## 832. 翻转图像

```
function flipAndInvertImage(A: number[][]): number[][] {
    let r = A.length, c = A[0].length;
    let ans: number[][] = new Array(r).fill(0).map(i => new Array(c).fill(0));
    for (let i = 0; i < r; i++) {
        for (let j = 0; j < c; j++) {
            ans[i][j] = A[i][c - j - 1] ^ 1;
        }
    }
    return ans;
};
```

#### 思路: 遍历一次矩阵，核心代码一行
