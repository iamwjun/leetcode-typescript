## 995. K 连续位的最小翻转次数

解法一 暴力 会超时
```
function minKBitFlips(A: number[], K: number): number {
    let ans = 0;
    for (let i = 0; i < A.length - K + 1; i++) {
        if (A[i] === 0) {
            ans++;
            let m = K;
            while (m > 0) {
                A[i + m - 1] = A[i + m - 1] ? 0 : 1;
                m--;
            }
        }
    }
    return A.indexOf(0) > -1 ? -1 : ans;
};
```
