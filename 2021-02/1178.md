## 1178. 猜字谜

### 解法一 超时
```
function findNumOfValidWords(words: string[], puzzles: string[]): number[] {
    let ans: number[] = [];
    const wSet = words.map(i => Array.from(new Set(Array.from(i))));
    for (const p of puzzles) {
        let total = 0;
        for (const w of wSet) {
            if(w.indexOf(p[0]) > -1 && w.every(i => p.indexOf(i) > -1)) total++;
        }
        ans.push(total);
    }
    return ans;
};
```

### 解法二
```
function flipAndInvertImage(A: number[][]): number[][] {
    return A.map(r => r.reverse().map(c => c ^ 1));
};
```

#### 思路: 遍历一次矩阵，核心代码一行
