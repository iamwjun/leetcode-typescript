## 153. 寻找旋转排序数组中的最小值

解法一 暴力，一次遍历
```
function findMin(nums: number[]): number {
    let pos = 0;
    for (let i = 1; i < nums.length; i++) {
        if(nums[i - 1] > nums[i]) {
            pos = i;
            break;
        }
    }
    return nums[pos];
};
```

解法二 二分查找
```
function findMin(nums: number[]): number {
    let l = 0, r = nums.length - 1;
    while (l < r) {
        const m = l + Math.floor((r - l) / 2);
        if (nums[m] < nums[r]) {
            r = m;
        } else {
            l = m + 1;
        }
    }
    return nums[l];
};
```
