## 338. 比特位计数

解法一 暴力
```
function countBits(num: number): number[] {
    let ans = new Array(num + 1).fill(0);
    for (let i = 0; i <= num; i++) {
        let total = 0, cur = i;
        while (cur > 0) {
            total += cur % 2;
            cur = cur >> 1;
        }
        ans[i] = total;
    }
    return ans;
};
```
