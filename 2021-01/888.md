## 888. 公平的糖果棒交换

```
function fairCandySwap(A: number[], B: number[]): number[] {
    const Alen = A.length,
        Blen = B.length,
        Atotal = A.reduce((a, b) => a + b, 0),
        Btotal = B.reduce((a, b) => a + b, 0);
    for (let i = 0; i < Alen; i++) {
        for (let j = 0; j < Blen; j++) {
            if ((Atotal - A[i] + B[j]) === (Btotal - B[j] + A[i])) {
                return [A[i], B[j]];
            }
        }
    }
};
```
#### 思路: 双指针
